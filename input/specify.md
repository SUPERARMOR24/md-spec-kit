# プロダクト名
EmojiLink (絵文字だけでつながるSNS)

# 目的
- 言語を超えてコミュニケーションできる場を作る
- 投稿もリアクションも絵文字のみで完結するシンプルな体験を提供する

# ペルソナ
- プログラミング未経験の高校生が自分の気分を共有したい
- 見て楽しい、触ってすぐ使えるシンプルなSNSを探している

# 技術の希望
- Node.js + React系（Vite/Nextどちらでも可、授業用にセットアップが速いもの）
- スタイルはシンプルで高校生が真似しやすいもの

# デモ条件
- ローカルで `npm install && npm run dev` 相当で起動できる
- 少量のデモ投稿が最初から表示される
