# ゴール
`output/spec.md` をもとに、最小機能の実装手順を `templates/template.plan.md` の構成で `output/plan.md` に日本語で出力する。

# 手順
1. スコープを再確認し、「絵文字投稿・タイムライン表示・絵文字リアクション・モックAPI」の完成をゴールに置く。
2. 大きな作業単位（例: 環境準備、UI実装、APIモック、結合テスト、デモデータ投入）に分解する。
3. 各ステップの目的・アウトプット・所要目安を簡潔に書く。
4. 体験授業向けに並行作業を避け、シリアルな流れにする。

# 出力フォーマット
- ファイル: `output/plan.md`
- `templates/template.plan.md` のセクションを埋める。
- 箇条書き中心、冗長な説明は省く。
