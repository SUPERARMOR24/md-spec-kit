# ゴール
`output/task.md` に徹って最小限の「絵文字だけでつながるSNS」のプロトタイプを実装し、進行状況を要約する。講師がデモ時に参照できるよう、日本語で稚的に記す。

# 手順
1. タスクを上から順に実行する。時間が不足したら「後回し」にタグ付けして明示する。
2. UIはモックデータ優先で構わない。投稿・リアクション・タイムライン表示が動くことを最優先。
3. 生成物（ファイル追加・更新）の概要を箇条書きでまとめる。

# 出力フォーマット
- 出力先: コンソールまたは記録用メモ（任意）
- 記述内容: 進行状況のサマリ、後回し項目、実行コマンド例
- すべて日本語。
